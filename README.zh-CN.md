# ❄️ DesktopSnow (幽灵版)

[🇺🇸 English (English)](README.md)

一个轻量级、全透明的 Windows 桌面下雪特效程序 (WPF)。

## 👻 幽灵模式说明 (Ghost Mode)

**注意：** 此程序设计为 **“幽灵模式”**。
* 运行后 **任务栏没有图标**。
* 运行后 **托盘区没有图标**。
* 它是完全隐形的，只在屏幕上下雪，不干扰日常操作。

## 🎮 快捷键控制

由于没有可视化界面，请务必记住以下控制键：

| 按键 | 功能 | 说明 |
| :--- | :--- | :--- |
| **F9** | 开/关 雪花 | 可以在 `config.txt` 中自定义按键 |
| **F12** | **彻底退出** | **紧急停止键，按下即杀进程** |

> 如果忘记快捷键，可以通过“任务管理器”结束 `DesktopSnow.exe` 进程。

## ⚙️ 配置文件

在程序同目录下创建 `config.txt` 文件进行配置（首次运行使用默认值）：

```ini
Mode=2            ; 1=按住显示, 2=切换显示(默认)
StartupShow=true  ; true=启动即下雪, false=启动等待
StartupDuration=5 ; 启动后自动下雪几秒钟
Key=120           ; 快捷键代码 (120=F9)
```
🚀 如何开机自启
1. 生成 exe 的快捷方式。
2. 按 Win + R 输入 shell:startup 打开启动文件夹。
3. 将快捷方式放入该文件夹。
4. 注意：建议在快捷方式属性中，将“起始位置”设置为 exe 所在目录，以确保正确读取配置。
